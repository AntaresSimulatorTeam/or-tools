operations_research::LocalSearchFilterManager Class Reference

Detailed Description

Filter manager: when a move is made, filters are executed to decide whether the solution is feasible and compute parts of the new cost.

This class schedules filter execution and composes costs as a sum.

overridevirtual

Returns true iff all filters return true, and the sum of their accepted objectives is smaller or equal to the target objective.

This target objective is: (objective_ == nullptr) ? kint64max : ((objective_ != delta->Objective()) ? objective_.Max() : min(objective_.Max(), delta->ObjectiveMax()))
